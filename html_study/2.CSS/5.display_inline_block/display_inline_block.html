HTML element의 display라는 속성 2가지 : Block / Inline

Block

div element는 block element. 항상 새로운 라인(줄)부터 시작함. 전체 가로 폭(width)을 100% 차지한다.

Paragraph element도 block element.

Inline

a element는 Inline element. 새로운 줄부터 시작하지 않고 자신이 필요한 영역만큼만 차지한다.

p element 내부에 중첩된(neted) span element는 자신의 영역만 차지함.
